# Project: Home credit scoring model

This project is an update of the project "Implement a scoring model" completed in 2022 during my Data Scientist training at OpenClassrooms in parternship with CentralSupélec.
The purpose is to implement and deploy an interactive decision support dashboard that calculates the probability of a customer repaying his loan. The dashboard serves as a decision-support tool, enabling businesses to make data-driven decisions to grant or refuse a credit application.

The machine learning models are trained on the following dataset: [Home Credit Default Risk](https://www.kaggle.com/c/home-credit-default-risk/data/)



# Projet : Modèle de scoring de crédit à la consommation

Ce projet est une mise à jour du projet "Implémentez un modèle de scoring" réalisé en 2022 dans le cadre de ma formation de Data Scientist chez OpenClassrooms en partenariat avec CentralSupélec.
L’objectif est de concevoir et déployer un tableau de bord interactif d'aide à la décision permettant de calculer la probabilité qu'un client rembourse son prêt. Le tableau de bord fonctionne comme un outil d’aide à la décision, permettant aux entreprises de prendre des décisions éclairées, basées sur les données, pour accorder ou refuser une demande de crédit.

Les modèles de Machine Learning sont entraînés sur le jeu de données suivant : [Risque de défaut de paiement pour le crédit à la consommation](https://www.kaggle.com/c/home-credit-default-risk/data/)
